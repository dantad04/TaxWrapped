# Australian Budget Wrapped Agent Notes

- Follow `docs/codex-board/ONE_TICKET_RULE.md`.
- Work on one ticket only. Stop after handoff unless explicitly prompted to continue.
- Income must remain client-side only. Do not store it in localStorage, sessionStorage, cookies, a database, analytics, or server logs.
- Do not add analytics.
- This project uses Next.js, TypeScript, Tailwind CSS, Vitest, and Playwright.
- This scaffold uses a current Next.js release with breaking changes from older versions. Read relevant local Next.js docs in `node_modules/next/dist/docs/` before changing framework conventions.
