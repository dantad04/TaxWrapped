# Ticket 005: Design System

Status: Human Review

## Goal

Create a small Australian Budget Wrapped visual system that supports the story flow without copying Tax Wrapped.

## Allowed Scope

- Define reusable typography, colour, spacing, and component conventions.
- Add shared UI primitives only where they remove real duplication.
- Keep the design mobile-first and accessible.
- Add tests or snapshots where useful for regressions.

## Non-goals

- Do not redesign unrelated product logic.
- Do not add final story-card polish, charts, animations, or sharing.
- Do not use Tax Wrapped's exact name, UI, copy, layout, animations, colours, illustrations, or assets.
- Do not add analytics or persistence.

## Acceptance Commands

- `npm run validate`
- `npm run test:e2e`
- `git diff --check`

## Visual Acceptance

The app has a coherent original visual identity, readable mobile layouts, clear focus states, and no overlapping text or controls.

## Handoff Requirements

Report files changed, design conventions introduced, commands run, validation result, and risks. Stop after this ticket.
